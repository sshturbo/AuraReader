Politica de Privacidade

Aura Reader | Ultima atualizacao: 07/03/2026

Esta Politica de Privacidade descreve como o Aura Reader coleta, usa, armazena e processa dados quando voce utiliza o site e a extensao de navegador para abrir PDFs, extrair texto, realizar leitura em voz alta e usar recursos opcionais de apoio com inteligencia artificial.

Resumo: o Aura Reader nao vende dados de usuarios. Os dados sao usados apenas para abrir PDFs, extrair texto, permitir a leitura em voz alta, manter preferencias locais e executar recursos opcionais de IA solicitados pelo proprio usuario.

1. Dados que podem ser processados

  • URLs de documentos PDF abertos pelo usuario.
  • Conteudo textual extraido de arquivos PDF processados no leitor.
  • Historico recente de PDFs abertos pela extensao.
  • Preferencias locais, como tema, voz selecionada e velocidade de leitura.
  • Chave de API informada pelo usuario para ativar recursos opcionais de IA.
  • Mensagens digitadas pelo usuario ao utilizar o modo de estudo com chat, quando esse recurso for usado.

2. Como os dados sao usados

  • Abrir e exibir documentos PDF no leitor.
  • Extrair o texto das paginas para leitura e acessibilidade.
  • Converter o conteudo em narracao por voz usando recursos do navegador.
  • Salvar configuracoes locais para melhorar a experiencia do usuario.
  • Executar recursos opcionais de IA, como reescrita acessivel ou apoio ao estudo, quando solicitados pelo usuario.

3. Uso de servicos de terceiros

Quando o usuario ativa recursos de IA e informa uma chave de API valida, partes do conteudo do PDF, assim como mensagens enviadas no chat de estudo, podem ser transmitidas ao provedor de IA configurado apenas para executar a funcionalidade solicitada. O tratamento desses dados tambem pode estar sujeito a politica de privacidade do provedor externo utilizado.

4. Armazenamento local

O Aura Reader pode armazenar localmente no navegador informacoes como preferencias de leitura, historico recente e chave de API fornecida pelo usuario. Esses dados sao mantidos no proprio navegador do usuario e nao sao vendidos nem compartilhados para fins alheios ao funcionamento do produto.

5. Compartilhamento de dados

O Aura Reader nao vende dados pessoais e nao transfere dados de usuarios a terceiros fora dos casos necessarios para cumprir o objetivo principal do produto, como o processamento opcional por um provedor de IA solicitado pelo usuario.

6. Seguranca

Sao adotadas medidas razoaveis para reduzir riscos no processamento de dados. Ainda assim, nenhum sistema de armazenamento ou transmissao e totalmente isento de risco, especialmente ao lidar com servicos externos.

7. Direitos do usuario

O usuario pode remover historico e preferencias locais limpando os dados do navegador ou removendo a extensao. Tambem pode deixar de usar recursos opcionais de IA a qualquer momento, removendo a chave de API ou nao ativando essas funcoes.

8. Alteracoes nesta politica

Esta politica pode ser atualizada para refletir mudancas no produto, nas integracoes ou em exigencias legais. A versao mais recente ficara sempre disponivel nesta mesma pagina.

9. Contato

Para solicitacoes relacionadas a privacidade, substitua este trecho pelo seu canal oficial de contato, como e-mail de suporte ou pagina institucional antes de publicar em producao.
